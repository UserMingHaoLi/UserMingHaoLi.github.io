null

null 关键字是表示不引用任何对象的空引用的文字值

null 是引用类型变量的默认值

但是null不是引用类型,他只是表示该变量没有任何值.

true和false

bool 类型关键字是 .NET System.Boolean 结构类型的别名，它表示一个布尔值，可为 true 或 false

可使用 true 和 false 文本来初始化 bool 变量或传递 bool 值

default

  • 指定 switch 语句中的默认标签。
  • 作为 default 默认运算符或文本生成类型的默认值。
